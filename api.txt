接收：
POST /api/passage/fetch：获取用户收藏的文章
	page: number >=1 页数
	step: number >=1 步长
	userId: String 用户名（测试用户名请见数据库）
		
POST /api/passage/collect：用户收藏文章
	title: string 文章名称
	loc: json 定位信息
	userId: String 用户名（测试用户名请见数据库）
		
POST /api/passage/del：用户删除文章
	id: number 文章id
		

POST /api/user/login：用户登录
	userId：String 用户名（测试用户名请见数据库）
	password ：string 用户密码（sha256加密字符串）

POST /api/user/check：用户登录状态检查
	userId：String 用户名（测试用户名请见数据库）
	base：String 用户身份标识串

POST /api/user/logout：用户登出
	userId：String 用户名（测试用户名请见数据库）
	base：String 用户身份标识串
	
返回：
1、 成功返回：
{
	"code": 0,
	"results": 结果
}
2、 失败返回：
{
	"code": -1,
	"msg": 错误状态码
}
